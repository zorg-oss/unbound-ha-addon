## Installation

1. Add repository to Home Assistant

   [![Add repository](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https://github.com/zorg-oss/unbound-ha-addon)

   Or manually: **Settings** > **Add-ons** > **Add-on Store** > **⋮** > **Repositories** > Add `https://github.com/zorg-oss/unbound-ha-addon`

2. Search "Unbound" in the Add-On store and install
3. Start the addon and open the Web UI to configure
4. location for a custom unbound.conf is /homeassistant/unbound/unbound.conf  
   verify the log the your custom unbound.conf is in use
